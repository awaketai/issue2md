# institution

经常需要在本地归档或者引GitHub上的Issue和Pull Request。手动复制粘贴太麻烦了，我想要做一个命令行工具，给它一个URL，它就能自动帮我把那个Issue
或PR的所有内容（标题、正文、评论等）转换成一个格式漂亮的Markdown文件保存下来

目前处在模糊意图阶段

+ “所有内容”具体指什么？Reactions要不要？用户链接要不要？
+ 命令行参数应该怎么设计？需要支持哪些Flag？
+ 除了CLI，未来是否考虑支持web界面？
